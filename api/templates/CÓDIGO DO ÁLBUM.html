OLIMPÍADA UNIVERSITÁRIA UFU 2025
Os jogadores de linha são avaliados por seis atributos principais: PAC - Ritmo; SHO - Finalização; PAS - Passe; DRI - Drible; PHY - Físico; DEF - Defesa. O peso desses atributos muda de acordo com a posição em campo.
Para goleiros, os seis atributos são diferentes: CHU - Chute; REF - Reflexo; MAN - Manejo; ELA - Elasticidade; VEL - Velocidade; POS - Posicionamento. Para eles, todos os atributos têm o mesmo peso no cálculo.
Overall é média ponderada dos atributos que representa o nível geral do jogador.
